# SkogAI Project

This project contains various components and scripts organized under the `.skogai` directory. Below is a brief overview of the main directories and their purposes:

## Directories

- **aider**: Contains configuration and history files related to the Aider tool.
- **docs**: Documentation files including changelogs, setup instructions, and test documentation.
- **tests**: Test scripts and related files for validating the functionality of the project.

## Notable Files

- **run-tests.zsh**: Script to execute all tests.
- **test_load_modules.zsh**: Zsh script to test module loading functionality.
- **test_load_modules.py**: Python script to test module loading functionality.
